# To-Do List Application (React + Vite)

This is a simple To-Do List application built using **React** and **Vite**.  
The application allows users to add, edit, delete, and mark tasks as completed.

This project was created as part of an academic assignment using only React
functional components, state, and props.

## Features

- Add new tasks  
- Edit existing tasks  
- Mark tasks as completed using a check button  
- Delete tasks using a trash icon  
- Simple animations for better user experience  
- Clean and user-friendly interface  

## Technologies Used

- React (Functional Components)
- Vite
- JavaScript
- CSS
